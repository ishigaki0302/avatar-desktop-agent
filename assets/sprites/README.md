# Sprites

PNG ファイルをここに配置してください。
ファイルがなくても、アバターはプレースホルダー（色付き円）で起動します。

## 命名規則

| ファイル名 | 説明 |
|-----------|------|
| `neutral_open.png` | 表情: neutral, 口: 開 |
| `neutral_close.png` | 表情: neutral, 口: 閉 |
| `happy_open.png` | 表情: happy, 口: 開 |
| `happy_close.png` | 表情: happy, 口: 閉 |
| `sad_open.png` | … |
| `sad_close.png` | … |
| `angry_open.png` | … |
| `angry_close.png` | … |
| `surprised_open.png` | … |
| `surprised_close.png` | … |
| `confused_open.png` | … |
| `confused_close.png` | … |
| `motion_nod_0.png` | nod モーション フレーム0 |
| `motion_nod_1.png` | … |
| `motion_nod_2.png` | … |
| `motion_wave_0.png` | wave モーション フレーム0 |
| … | … |

## 推奨サイズ

320×320px、PNG（透過可）。
